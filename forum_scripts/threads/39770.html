That is where you go wrong... If you call the jsp directly your commandObject isn't going to be setup neither is your referenceData. You have to make sure you pass through the dispatcher servlet.

try http://localhost:8080/myapp/statusmsg.html that should pass you through the controller. Spring will interprete the GET request as an initial request and setup the command object, reference data etc. The next call (POST) will pass through and the onSubmit method will be called.
