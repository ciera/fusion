Please use [ code][/ code] (square brackets instead of < >).

The problem is you are setting your own model which effectivly removes the current and correct model of your page. The model is on the errors instance and can be retrieved with the getModel method.

Also why are you retrieving a new instance of the profile instead of working with the one that is being submitted?

Your on submit method should look something like this

Code:
public ModelAndView onSubmit(HttpServletRequest request, HttpServletResponse response, Object command, BindException errors) throws ServletException {
	i++;
	logger.info("Value of i--->" + i);
	logger.info("Enetring onSubmit Method of LogonFormController");
	
	Profile credentials = (Profile) command;
	logger.info("after Command Bean:"+ credentials);
	
	List profileList = getCommandBeanManager().getCredentialsDAO().getProfile();
	logger.info("profileList"+ profileList);
	
	String FirstName = (String) profileList.get(0);
	String LastName = (String) profileList.get(1);
	String Gender = (String) profileList.get(2);
	logger.info("FirstName:"+ FirstName);
	logger.info("LastName:"+ LastName);
	logger.info("Gender:"+ Gender);
	
	Map model = errors.getModel();
	return new ModelAndView("logoncommand", errors.getModel());
}
