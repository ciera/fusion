You cann't set properties directly on the command object from javascript.

You could however set a value on a hidden input field which is later bound to your commandobject.

In your jsp
Code:
<form:hidden path="action"/>
In javascript

Code:
function saveProject() {
 document.formname.action.value='saveProject';
}
This way you can set values which are later bound to the command object.

However seeing your javascript it seems as if you want to use the action property on the command object to identify what to do in the Controller. You can also use different submit buttons for that. Only the submit button pressed will be included in the request parameters.

In your html

Code:
<input type="submit" name="saveProject" value="Save"/>
<input type="submit" name="removeProject" value="Remove"/>
In your code

Code:
if (WebUtils.hasSubmitParameter(request, "saveProject") ) {
 doSave();
} else if (WebUtils.hasSubmitParameter(request, "removeProject") ) {
 doRemove();
}
