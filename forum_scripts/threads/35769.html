	}

The error is in your jsp.

Code:

Replace it with

Code:
<option value="">--Please choose--</option>

Ok good. From the previous posts it looked like you had set up, or were going to set up an implementation of Validator. But you don't need both the Validator and onBindAndValidate. So you did it right.

I like to use Validator implementation for all of my forms rather than use onBindAndValidate. I do this for consistency and to increase understandability and modifiability. When it comes time to make a change later on, I know that all of my validation is done in Validators, which are packaged together, so I know right where to go.

I think its ok to use onBindAndValidate, just make sure you are consistent. I hope that works out for you.
