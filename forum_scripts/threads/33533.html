If you are using SimpleFormController you can just append an error in one of the onSubmit callbacks (normally you would only implement one), e.g.

Code:
protected ModelAndView onSubmit(Object command, BindException errors) {
    try {
        // do stuff with command
    } catch (DataAccessEcxeption e) {
        errors.reject("wrong", "Wrong Dummy!");
    }
    return super.onSubmit(command, errors);
}
If you aren't using SimpleFormController you can probably do the same thing by copying some of the code from there into your own controller.
