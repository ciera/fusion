The url is still there in the browser, as soon as someone presses the back button in the browser the old url with the not existing flowId is being presented. Basicly there is nothing you can do. You can only improve the user experience and not show the error.

You can create a HandlerExceptionResolver which handles the specific exception nicely, i.e. redirecting to front page or restart the flow.

We created a SimpleFlowHandlerMapping for that. We can map our flows to certain urls while still using 1 controller. It kind of works like the SimpleUrlHandlerMapping only instead of mapping it to a Controller we map it to a flow. So url1.flow maps to flow1 and url2.flow maps the flow2. Now if an Exception occurs we have the original URL url1.flow or url2.flow and we simply redirect to that flow WITHOUT the flowId as the parameter. Works like a charm.

Edit: code is to long to post here .


For this to work we first need a different FlowExecutorArgumentHandler.
Code:
/**
 * ArgumentHandler implementation which also chechs the attributes of the request for a 
 * flowId. Normally being used in combination with {@link SimpleFlowHandlerMapping} which adds
 * the flowId to the attributeMap of the request.
 * 
 * @author M. Deinum (487)
 * @since 30-11-2006
 *
 */
public class RequestAttributeAndParameterFlowExecutorArgumentHandler extends
        RequestParameterFlowExecutorArgumentHandler {
    
    private final Logger logger = LoggerFactory.getLogger(RequestAttributeAndParameterFlowExecutorArgumentHandler.class);
    
    @Override
    public boolean isFlowIdPresent(ExternalContext context) {
        if (context.getRequestMap().contains(getFlowIdArgumentName())) {
            return true;
        }        
        return super.isFlowIdPresent(context);
    }
    
    @Override
    public String extractFlowId(ExternalContext context)
            throws FlowExecutorArgumentExtractionException {
        String flowId = context.getRequestMap().getString(
                getFlowIdArgumentName());
        if (flowId == null) {
            logger.debug("Retrieving flowId from request parameters.");
            flowId = context.getRequestParameterMap().get(getFlowIdArgumentName());
        }
        flowId = applyDefaultFlowId(flowId);
        
        if (logger.isDebugEnabled()) {
            logger.debug("Retrieved flowId from request '"+flowId+"'");
        }
        
        if (!StringUtils.hasText(flowId)) {
            throw new FlowExecutorArgumentExtractionException(
                    "Unable to extract the flow definition id parameter: make sure the client provides the '"
                            + getFlowIdArgumentName()
                            + "' parameter as input or set the 'defaultFlowId' property; "
                            + "the parameters provided in this request are: "
                            + StylerUtils.style(context
                                    .getRequestParameterMap()
                                    + ". The attributes provided are: "
                                    + StylerUtils
                                            .style(context.getRequestMap())));
        }
        return flowId;
    }
}
Configur the FlowController to use it
Code:
<bean id="flowController" class="org.springframework.webflow.executor.mvc.FlowController">
	<property name="flowExecutor" ref="flowExecutor"/>
	<property name="cacheSeconds" value="5"/>
	<property name="argumentHandler">
		<bean class="nl.ctb.common.webflow.executor.support.RequestAttributeAndParameterFlowExecutorArgumentHandler"/>
	</property>
</bean>
The reason we created this is we didn't like the flowId stuff on the url and this was a nicely added bonus .

The SimpleFlowHandlerMapping

Code:
public class SimpleFlowHandlerMapping extends AbstractHandlerMapping implements
        InitializingBean {
    private Logger logger        = LoggerFactory
                                                   .getLogger(SimpleFlowHandlerMapping.class);
    private FlowController   flowController;
    private UrlPathHelper    urlPathHelper = new UrlPathHelper();
    private PathMatcher      pathMatcher   = new AntPathMatcher();
    private final Map        handlerMap    = new HashMap();
    private final Map        urlMap        = new HashMap();

    public void setFlowController(final FlowController flowController) {
        this.flowController = flowController;
    }

    @SuppressWarnings("unchecked")
    public void setMappings(Map mappings) {
        urlMap.putAll(mappings);
    }

    public void setAlwaysUseFullPath(boolean alwaysUseFullPath) {
        this.urlPathHelper.setAlwaysUseFullPath(alwaysUseFullPath);
    }

    public void setUrlDecode(boolean urlDecode) {
        this.urlPathHelper.setUrlDecode(urlDecode);
    }

    public void setUrlPathHelper(UrlPathHelper urlPathHelper) {
        this.urlPathHelper = urlPathHelper;
    }

    public void setPathMatcher(PathMatcher pathMatcher) {
        Assert.notNull(pathMatcher, "PathMatcher must not be null");
        this.pathMatcher = pathMatcher;
    }

    protected Object getHandlerInternal(HttpServletRequest request)
            throws Exception {
        String lookupPath = this.urlPathHelper.getLookupPathForRequest(request);
        logger.debug("Looking up handler for [{}]", lookupPath);
        return lookupHandler(lookupPath, request);
    }

    protected Object lookupHandler(String urlPath, HttpServletRequest request) {
        // direct match?
        String flowId = (String) this.handlerMap.get(urlPath);
        if (flowId == null) {
            // pattern match?
            String bestPathMatch = null;
            for (Iterator it = this.handlerMap.keySet().iterator(); it
                    .hasNext();) {
                String registeredPath = (String) it.next();
                if (this.pathMatcher.match(registeredPath, urlPath)
                        && (bestPathMatch == null || bestPathMatch.length() <= registeredPath
                                .length())) {
                    bestPathMatch = registeredPath;
                }
            }
            if (bestPathMatch != null) {
                flowId = (String) this.handlerMap.get(bestPathMatch);
                exposePathWithinMapping(this.pathMatcher
                        .extractPathWithinPattern(bestPathMatch, urlPath),
                        request);
            }
        } else {
            exposePathWithinMapping(urlPath, request);
        }
        if (flowId != null) {
            logger.debug("Found flow '{}' for urlPath '{}'", flowId, urlPath);
            
            request.setAttribute(flowController.getArgumentHandler()
                    .getFlowIdArgumentName(), flowId);
            return flowController;
        }
        return null;
    }

    protected void exposePathWithinMapping(String pathWithinMapping,
            HttpServletRequest request) {
        request.setAttribute(
                HandlerMapping.PATH_WITHIN_HANDLER_MAPPING_ATTRIBUTE,
                pathWithinMapping);
    }

    protected void registerHandler(String[] urlPaths, String beanName)
            throws BeansException, IllegalStateException {
        Assert.notNull(urlPaths, "URL path array must not be null");
        for (int j = 0; j < urlPaths.length; j++) {
            registerHandler(urlPaths[j], beanName);
        }
    }

    protected void registerHandler(String urlPath, Object handler)
            throws BeansException, IllegalStateException {
        Assert.notNull(urlPath, "URL path must not be null");
        Assert.notNull(handler, "Handler object must not be null");
        Object mappedHandler = this.handlerMap.get(urlPath);
        if (mappedHandler != null) {
            throw new IllegalStateException("Cannot map handler [" + handler
                    + "] to URL path [" + urlPath
                    + "]: There is already handler [" + mappedHandler
                    + "] mapped.");
        }
        if (urlPath.equals("/*")) {
            setDefaultHandler(handler);
        } else {
            this.handlerMap.put(urlPath, handler);
            if (logger.isDebugEnabled()) {
                logger.debug("Mapped URL path [" + urlPath + "] onto handler ["
                        + handler + "]");
            }
        }
    }

    public void initApplicationContext() throws BeansException {
        super.initApplicationContext();
        registerHandlers(this.urlMap);
    }

    protected void registerHandlers(Map urlMap) throws BeansException {
        if (urlMap.isEmpty()) {
            logger.warn("No 'mappings' set on SimpleUrlHandlerMapping");
        } else {
            Iterator it = urlMap.keySet().iterator();
            while (it.hasNext()) {
                String url = (String) it.next();
                Object handler = urlMap.get(url);
                // Prepend with slash if not already present.
                if (!url.startsWith("/")) {
                    url = "/" + url;
                }
                registerHandler(url, handler);
            }
        }
    }

    public void afterPropertiesSet() throws Exception {
        Assert.notNull(flowController,
                "FlowController must be set on SimpleFlowHandlerMapping.");
    }
}
The configuration
Code:
<bean id="flowMapping" class="nl.ctb.common.webflow.servlet.handler.SimpleFlowHandlerMapping">
	<property name="flowController" ref="flowController"/>
	<property name="mappings">
		<map>
			<entry key="/user-search.flow" value="search-user-flow"/>
			<entry key="/checkout.flow" value="checkout-flow"/>				
        </map>
    </property>
</bean>
And that is it.
