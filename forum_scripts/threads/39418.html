Which makes it clear why your code isn't working. The cutOffTime property is going to be injected AFTER the constructor is called so in your constructor it is going to be null. This is logical imho you can only inject stuff after you have an instance of your Controller, an instance is created by calling the constructor.

Move your code from the constructor to a public method named initialize() (for example) and add init-method="initialize" to your bean definition.
